# Calendar

The **Calendar** nav item (`/mediastack`) surfaces Sonarr, Radarr, Lidarr, Bazarr, and download-client activity inside the portal. Configure instances under **Settings → Media Stack**.

## Sections

| Section | Purpose |
| --- | --- |
| Release Calendar | Upcoming episodes and movies with availability state |
| Active Queue | Current Sonarr/Radarr/Lidarr-matched downloads and progress |
| Recent History | Imports, grabs, and service history |
| Month Navigation | Browse releases across months |
| ID Matching | Map content through IMDb, TMDB, and TVDB IDs |
| Bazarr Tools | Subtitle widgets and multi-instance subtitle actions |
| Download Status | qBittorrent, Transmission, BitTorrent, Deluge, and SABnzbd with source filters |

## Downloads page

There is also a dedicated **Downloads** nav item that merges every enabled download client into one status view, with Sonarr / Radarr / Lidarr / other filters. See [Integrations → Download Status](/guide/integrations#download-status).
